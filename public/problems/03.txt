#TITLE 03 - Book Library File Subsystem
#SUMMARY Task 03: File-based Data Storage for Adding and Removing Books

## Problem Overview
You are managing a small library and need to store your collection in a persistent text file. Your task is to develop a script that manages adding and removing books utilizing file I/O operations.

---

### Core Requirements
To complete this task, your Python script must implement the following:

1.  **Class Definition**: Create a `Book` class with a constructor (`__init__` method) that initializes the `name`, `author`, and `price` attributes.
2.  **File Management**:
    *   Create a function `add_book(book)` that takes a `Book` object and appends its details (e.g., as a comma-separated string) to a file named `library.txt`.
    *   Create a function `remove_book(book_name)` that reads all lines from `library.txt`, removes any book having a matching `book_name`, and overwrites the file with the updated data.
    *   Create a function `display_books()` that reads and prints all the books from `library.txt`.
3.  **Outcome Verification**: 
    *   Instantiate 3 `Book` objects.
    *   Add each of the 3 books using your `add_book` function.
    *   Call `display_books()` to list the books.
    *   Remove 1 book using `remove_book`.
    *   Call `display_books()` again to verify the specific book has been removed.

---

### Expected Output Structure
When executed, your program should display output formatted like the following:

```text
--- Library Books ---
Name: The Great Gatsby | Author: F. Scott Fitzgerald | Price: $10.99
Name: 1984 | Author: George Orwell | Price: $12.50
Name: To Kill a Mockingbird | Author: Harper Lee | Price: $8.99

Removing '1984'...

--- Library Books ---
Name: The Great Gatsby | Author: F. Scott Fitzgerald | Price: $10.99
Name: To Kill a Mockingbird | Author: Harper Lee | Price: $8.99
```

---

### Implementation Guide
> [!TIP]
> **File Modes**: Use `'a'` mode to append to a file, `'r'` to read from it, and `'w'` to overwrite the file content.

> [!IMPORTANT]
> When reading lines from the file, use `.strip()` to remove any trailing newline (`\n`) characters before splitting the string by a comma.
